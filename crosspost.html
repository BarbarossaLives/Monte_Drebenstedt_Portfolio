Business Tools/CrossPosting

This custom desktop application automates social media posting across multiple platforms from a single interface. Designed for creators and small business owners, it simplifies the repetitive task of posting the same content to multiple sites manually.

Key Features:

  • Crossposts to Ko-fi, Mastodon, Dev.to, Discord (up to 5 channels), and LinkedIn
  • Clean, modern GUI built with Python + TTKBootstrap
  • Image upload support and optional Ko-fi link embedding
  • Config system to save reusable posting templates
  • Extensible modular codebase with each platform handled in a separate module

Tech Stack:

  • Python (core logic)
  • TTKBootstrap (GUI framework)
  • Requests, dotenv (API interaction, config handling)
  • Modular architecture for clean code and easy upgrades

Use Case:Perfect for indie developers, creators, or social media managers who need to post regularly across platforms without the complexity of web-based schedulers or expensive SaaS tools.
