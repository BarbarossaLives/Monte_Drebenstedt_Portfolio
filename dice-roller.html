Dice Roller Code

This tool is part of a larger TTRPG toolkit built in Python. It allows users to roll standard dice types (d4, d6, d8, d10, d12, d20, d100) with optional modifiers. It is designed with flexibility in mind, supporting both standalone desktop GUI use and future integration with Discord bots or virtual tabletops.

The project has also been expanded into Godot, with the goal of incorporating it into a fully developed computer game. Both versions follow the same core logic and will serve as the foundation for a comprehensive suite of TTRPG tools.

The next phase of the project will involve creating a Discord bot to further enhance accessibility and integration with online gameplay environments.
